# Data Card: Thai Pump Radar AI-Ready Dataset

## 📝 ข้อมูลทั่วไป (Overview)
- **ชื่อชุดข้อมูล:** Thai Pump Radar Community Reports
- **แหล่งที่มา:** [thaipumpradar.com](https://thaipumpradar.com/) (Community-driven platform)
- **ประเภทข้อมูล:** รายงานสถานะน้ำมันแบบเรียลไทม์จากผู้ใช้งาน
- **รูปแบบไฟล์:** 
  - `radar_clean.parquet` (สำหรับการวิเคราะห์เชิงสถิติ)
  - `radar_clean.jsonl` (สำหรับการเทรน AI หรือใช้ในระบบ RAG)

## 🎯 วัตถุประสงค์ (Intended Use)
- ใช้ในการติดตามสถานะน้ำมันขาดช่วงในแต่ละพื้นที่เรียลไทม์
- ใช้เป็นฐานข้อมูลสำหรับ AI Agent เพื่อตอบคำถามผู้ใช้ (RAG)
- ใช้ในการวิเคราะห์แนวโน้ม (Trend Analysis) และการพยากรณ์ความต้องการน้ำมัน

## 🔍 โครงสร้างข้อมูล (Dataset Schema)
| ชื่อฟิลด์ (Field Name) | คำอธิบาย (Description) | ตัวอย่าง (Example) |
| :--- | :--- | :--- |
| `stationName` | ชื่อปั๊มน้ำมัน | ปั๊ม ปตท. สาขาสนามบิน |
| `brandId` | รหัสแบรนด์ (PTT, Shell, etc.) | `PTT` |
| `province` | จังหวัด | เชียงใหม่ |
| `district` | อำเภอ | เมือง |
| `diesel` | สถานะดีเซล (out, available, etc.) | `out` |
| `e20` | สถานะ E20 | `available` |
| `confidence` | คะแนนความเชื่อมั่น (0-1) | `0.95` |
| `createdAt` | เวลาที่รายงาน (Unix Timestamp) | `1710825600` |

## 🧪 กระบวนการทำความสะอาด (Cleaning & Validation)
- **ตัวกรองความเชื่อมั่น (Confidence Filter):** คัดเลือกเฉพาะรายงานที่มีค่า `confidence >= 0.7` เท่านั้น (ในเดโมใช้ 0.3 เพื่อแสดงผลข้อมูลใหม่)
- **การจัดการค่าว่าง:** แปลงสถานะที่ไม่มีข้อมูล (Unknown) เป็นค่ามาตรฐานเดียวกัน

## ⚠️ ข้อจำกัดและข้อควรระวัง (Limitations & Risks)
- **Community Sourced:** ข้อมูลเกิดจากรายงานของผู้ใช้ อาจมีความคลาดเคลื่อนหรือดีเลย์
- **Bias:** บางพื้นที่อาจมีรายงานมากกว่าพื้นที่อื่นเนื่องจากจำนวนผู้ใช้ในแอป
- **Temporal Nature:** สถานะน้ำมันเปลี่ยนแปลงได้ตลอดเวลา ข้อมูลมีความสดใหม่สูงแต่มีอายุการใช้งานสั้น
